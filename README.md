# Reinforcement Learning in Finance

This is an attempt of replicating the algorithms and results from the survey paper,
> Hambly, B., Xu, R., & Yang, H. (2023). *Recent Advances in Reinforcement Learning in Finance*. arXiv:2112.04553. https://doi.org/10.48550/arXiv.2112.04553

## Algorithms
 - Temporal Differential Learning
 - Q-Learning 
 - SARSA
 - Monte Carlo Plocy Gradient
 - Deep Q-Network (DQN)
